The crowdsourced mobile web application for people who drive a lot and need a restroom.

Built with React.

Drivers for services like Uber or Lyft often need to quick find a restroom in areas they don't know very well. This is where Gotta Go comes in! We devided all spots on three categories: public, restaurants, gas stations. Logged in users can add new location to database, leave comments, update account. Unregistered users can use aplication in view mode.

Technologies/tools used: HTML, CSS, Bloomer, React, Google Maps API, Mongoose, Passport, Express, Node.js, npm, Git/Github, Heroku, Trello, Slack

This application was created as a group project team of four devs for the UC Berkeley Extension Coding Boot Camp. Using React Native we are continuing build Native iOS/Android versions of the app.
